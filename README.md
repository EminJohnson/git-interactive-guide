# ⚡ Interactive Git Guide | دليل Git التفاعلي ⚡

## 🛠️ عن المشروع (About)
هذا المشروع هو **واجهة تفاعلية ذكية** مصممة لتبسيط مفاهيم Git المعقدة. بدلاً من البحث في القوائم الطويلة، يوفر التطبيق تجربة بصرية تعتمد على البيانات لتوجيه المطورين نحو الأوامر الصحيحة بناءً على "دورة حياة المشروع".

> **"Bridging the gap between Git complexity and developer productivity."**

## ✨ المميزات الرئيسية (Key Features)
*   📊 **Visual Analytics:** استخدام رسم بياني (Polar Area Chart) لتصنيف الأوامر ووزنها النسبي.
*   🔍 **Smart Filtering:** نظام بحث فوري يعتمد على الكلمات المفتاحية (JS Logic).
*   🎨 **Modern UI/UX:** تصميم "Glassmorphism" هادئ ومريح للعين باستخدام Tailwind CSS.
*   📱 **Responsive Design:** متوافق تماماً مع جميع الشاشات (Mobile-First).
*   📋 **One-Click Copy:** نظام تفاعلي لنسخ الأوامر مباشرة (قيد التطوير).

## 🚀 التكنولوجيا المستخدمة (Tech Stack)
*   **HTML5** (Semantic Structure)
*   **Tailwind CSS** (Modern Styling)
*   **Chart.js** (Data Visualization)
*   **JavaScript (ES6)** (Interactive Logic)

## 📂 هيكلية الكود (Code Philosophy)
بصفتي مهتم بـ **Clean Code**، تم بناء المشروع ليكون:
1.  **Modular:** البيانات منفصلة تماماً عن الواجهة (Data-Driven UI).
2.  **Lightweight:** لا يحتاج إلى مكتبات ثقيلة، يعتمد على CDN للأداء السريع.
3.  **Readable:** تسميات واضحة للمتغيرات وهيكلية منظمة تسهل التطوير المستقبلي.

## 📥 كيف تشغل المشروع؟
لا حاجة لعمليات تثبيت معقدة! فقط قم بتحميل المستودع وافتح ملف `index.html` في متصفحك:

```bash
git clone https://github.com
cd git-interactive-guide
open index.html
```
